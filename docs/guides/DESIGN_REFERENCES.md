## Color Palette Reference

Based on the inspirations and current design:
- **Primary Gradient**: `#667eea` → `#764ba2`
- **Glass Background**: `rgba(255, 255, 255, 0.08)` to `rgba(255, 255, 255, 0.15)`
- **Text Colors**: 
  - Primary: `#FFFFFF` (white) or `#333333` (dark)
  - Secondary: `#666666` or `#94A3B8`
- **Borders**: `rgba(255, 255, 255, 0.1)` to `rgba(255, 255, 255, 0.2)`
- **Shadows**: `rgba(0, 0, 0, 0.1)` to `rgba(0, 0, 0, 0.3)` with purple tints

## Typography Reference

- **Font**: DM Sans (Google Fonts)
- **Weights**: 400 (regular), 500 (medium), 600 (semi-bold), 700 (bold)
- **Sizes**: 12px (small), 14px (body), 16px (base), 18px (large), 24px+ (headings)
- **Line Height**: 1.5 for body, 1.4 for headings

## Animation Reference

- **Timing**: 150ms (fast), 200-300ms (normal), 400-500ms (slow)
- **Easing**: `ease-in-out` (default), `cubic-bezier(0.4, 0, 0.2, 1)` (smooth)
- **Effects**: Fade, slide, scale, bounce (subtle)

## Best Practices from References

1. **Consistency**: Maintain consistent design language throughout
2. **Clarity**: Prioritize readability and usability
3. **Depth**: Use layers, shadows, and elevation for depth
4. **Smoothness**: All animations should be smooth and purposeful
5. **Premium Feel**: Every element should feel polished and premium
6. **Accessibility**: Maintain high contrast and clear focus states
7. **Responsiveness**: Design should work beautifully on all screen sizes
